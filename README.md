# HostelPro

Modern Hostel Management System built with Electron + React + TypeScript + SQLite.

## Features

- Dashboard
- Boarder Management
- Room Management
- Payment Tracking
- Reports
- Settings
- Backup & Restore
- SQLite Persistent Database

## Tech Stack

- Electron
- React
- TypeScript
- Vite
- Zustand
- SQLite

## Run

```bash
npm install
npm run electron:dev
